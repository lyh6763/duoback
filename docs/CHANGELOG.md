## [1.3.0] - 2026-01-30

### ✨ Added
- 제품 공통 데이터 소스 `js/products-data.js` 도입  
  - 이유: 제품 데이터 중복 제거 및 유지보수 단일화
- 제품 리스트 데이터 기반 렌더링 전환  
  - 이유: 하드코딩 제거 및 확장성 확보
- 리스트 뷰 확장 정보 (요약/하이라이트/액션 버튼)  
  - 이유: 리스트 탐색 시 정보 밀도 강화
- 리스트 뷰 컬러 칩 + 컬러별 이미지 매핑  
  - 이유: 옵션 선택 경험 개선 및 시각 피드백 강화
- URL 카테고리 필터(`?category=`) + 빈 상태 UI  
  - 이유: 카테고리 탐색 및 예외 케이스 대응

### 🔧 Scripts
- 검색/제품 상세/리스트 기능이 공통 데이터 사용하도록 통합  
  - 이유: 데이터 일관성 보장
- 리스트에서 빠른 장바구니 담기 + 색상 선택 연동  
  - 이유: 구매 플로우 단축

## [1.2.0] - 2026-01-28

### 🧹 Cleanup
- `images/brand`, `images/hero`, `images/icons`, `images/logo` 폴더 제거  
  - 이유: 비어 있거나 실제 페이지에서 참조되지 않아 불필요한 자산 정리

### ♿ Accessibility
- 모바일 메뉴 버튼 `aria-expanded` 동기화  
  - 이유: 상태 전달 정확도 개선
- FAQ 아코디언 ARIA 속성 추가  
  - 이유: 질문/답변 관계 전달
- 탭 UI ARIA 동기화  
  - 이유: 키보드/스크린리더 탐색 보장
- 검색 모달 `aria-live`/`role="status"` 추가  
  - 이유: 결과 변화 알림
- 폼 `autocomplete` 보강  
  - 이유: 입력 편의 개선
- 스펙 테이블 `caption`/`th scope` 추가  
  - 이유: 표 구조 이해도 향상

### 🔗 Links & Meta
- 푸터 앵커 타깃(`#privacy`, `#terms` 등) 대응 요소 추가  
  - 이유: 죽은 링크 제거
- OG/Twitter 메타 URL 및 이미지 로컬 기준으로 정리  
  - 이유: 배포 전 환경에서도 깨짐 최소화

### 🎨 Design (Tone Update)
- 테크/프리미엄 톤 컬러 및 배경 그리드 적용  
  - 이유: 제품 정밀성/프리미엄 인상 강화
- 타이포그래피 시스템 교체  
  - 이유: 기술적 정밀감 + 고급 인상 동시 확보
- 메인 히어로 레이아웃 재구성  
  - 이유: 구조적 리듬과 정보성 강화
- 서브 페이지 헤더 eyebrow 추가  
  - 이유: 톤 통일 및 시각적 계층 강화
